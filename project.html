Introduction

  • A college library management is a project that manages and stores books information electrically according to student needs.
  • The system is help both students and library manager to keep a contain track of all the books available in the library. It allows both the admin and the student to search for the desired book.
  • This project contains Available book , List of Available book , list of Issue book , Late book, Add book ,Remove book ,issue book , Search book by Name , Number , Author Name , and main menu of the library.

      Purpose Collect :
    • Collect , store ,organize ,retrieve and make available the information sources to available the information users.
      Objectives:
    • To operate library with efficiency and at reduced cost
      Goal
    • Manage the details of Student , Books , Issues , Librarian , Member.
